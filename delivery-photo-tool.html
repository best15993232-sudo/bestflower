사용 방법

  1. 1메인페이지 (bestflower.co.kr)를 엽니다.
  2. 2페이지에서 Ctrl + U (또는 우클릭 → 페이지 소스 보기)를 누릅니다.
  3. 3Ctrl + A로 전체 선택 후 Ctrl + C로 복사합니다.
  4. 4아래 입력창에 Ctrl + V로 붙여넣고 "사진 추출" 버튼을 클릭합니다.
  5. 5추출된 사진을 확인하고 "코드 생성" 버튼을 클릭합니다.
  6. 6생성된 코드를 index.html에 붙여넣습니다.
